Dieses Projekt verfügt über zwei verschiedene Menüs. In der Datei risikotest.java können Sie das gewünschte Menü deaktivieren. Beide sind standardmäßig aktiviert. Sobald Sie das erste Menü schließen, öffnet sich das zweite. Dort können Sie auf „Risiko“ und anschließend auf „Risiko hinzufügen“ klicken. Sie werden nach dem Namen des Risikos, seiner Eintrittswahrscheinlichkeit und den anfallenden Kosten gefragt. Anhand dieser Werte berechnet das Programm automatisch die Rückstellungen und ordnet das Risiko einer von drei Kategorien zu: akzeptables Risiko, inakzeptables Risiko (bei dem Sie einen Schadensfall angeben müssen) und extremes Risiko (bei dem Sie sowohl einen Schadensfall als auch einen Versicherungsbeitrag angeben müssen).
